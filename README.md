# Giphy Store

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run start
```

### Compiles and minifies for production

```
npm run build
```

2. #### apollo-link-rest from appolo to call rest apis

3. #### React-Router V.6 for navigation.
